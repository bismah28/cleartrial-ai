# ClearTrial AI

![ClearTrial AI Screenshot](your-screenshot-file-name.png)

## Overview

ClearTrial AI is a web application that helps users discover recruiting clinical trials and understand eligibility requirements in plain English.

The platform retrieves real-time clinical trial information from ClinicalTrials.gov and presents study details, eligibility criteria, match scores, and direct links to official trial pages.

## Features

* Search clinical trials by condition and location
* Retrieve live data from ClinicalTrials.gov
* View recruiting study information
* Read simplified eligibility summaries
* Generate match scores
* Open official study pages directly from the application
